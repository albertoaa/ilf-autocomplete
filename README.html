ILF Prueba Javascript - React

Cree un componente de autocompletar que permita a los usuarios ingresar un término de búsqueda para buscar y seleccionar un estado de EE. UU.

Esta es una prueba de funcionalidad, no necesita crear un diseño Pixel Perfect.

El propósito de esta prueba es examinar su capacidad para escribir código y estructurar un pequeño componente React.

Ya se importó un archivo de css, así como algunos otros archivos necesarios, por lo que está listo para comenzar a escribir su código.

client / js y client / sass son los directorios en los que trabajará.

Especificaciones funcionales

Ver component-wireframe.pdf para los wireframes de diseño.

  • Comience a buscar después de ingresar 2 o más caracteres en el campo de entrada
  • Al buscar, debe presionar el punto final de API proporcionado a continuación y devolver los nuevos resultados en la lista
  • Navegue hacia arriba y hacia abajo a través de los resultados con las teclas de flecha: mientras navega, el valor de entrada se actualizará con la etiqueta apropiada
  • Seleccione un resultado presionando "regresar" o haciendo clic con el mouse al pasar sobre un resultado
  • Al seleccionar un resultado, la lista de resultados debería desaparecer
  • Muestra un máximo de 5 elementos en el resultado antes de que el scroll vertical sea visible

Herramientas de compilación necesarias

  • Node - >= 6.1.x <=9.x.x
  • Webpack - Instalar vía terminal: sudo npm install -g webpack@3.10.0

Fameworks y Herramientas requeridas:

  • React
  • SASS

Instrucciones

  • Abra la terminal y navegue hasta la raíz del proyecto
  • Ejecutar: npm install
  • Una vez completada la instalación, ejecute: webpack
  • Después de la compilación, abra su navegador y navegue a localhost:3000

Cuando realice un cambio en un archivo Javascript o SCSS en la carpeta cliente , se compilará automáticamente, tendrá que actualizar su navegador manualmente para ver el cambio.

API

localhost:3000/api/states?term=<search term>
